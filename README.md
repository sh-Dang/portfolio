# 🚀 SeHyung's Portfolio

## 💡 프로젝트 소개
이 프로젝트는 **React**로 제작한 개인 포트폴리오 사이트입니다.  
포트폴리오를 통해 저의 기술 스택, 프로젝트 경험, 그리고 연락처 정보를 확인할 수 있습니다.

---

## 🛠 기술 스택
- **Frontend:** ⚛️ React, JavaScript, HTML, CSS  
- **배포:** 🌐 GitHub Pages  
- **CI/CD:** 🤖 GitHub Actions

---

## ✨ 주요 기능
- 📱 반응형 디자인 (PC / 모바일)  
- 📂 프로젝트 및 기술 스택 소개  
- ⚡ GitHub Actions를 통한 자동 배포 (커밋 시 GitHub Pages 업데이트)

## ⚙️ CI/CD
- **자동화 배포:** main 브랜치에 push 시 GitHub Actions 실행 → 자동으로 GitHub Pages에 배포  
- **작업 흐름**
  1. 📝 코드 push → GitHub Actions 실행  
  2. 🏗 React 프로젝트 빌드  
  3. 📦 `gh-pages` 브랜치에 빌드 결과 배포
